# DriveGuard – Smart Driver Safety System 🚗⚠️

DriveGuard is a smart embedded safety system designed to detect driver drowsiness in real time using an IR eye blink sensor and Arduino Nano.

If prolonged eye closure is detected, the system automatically:
- Activates buzzer alerts
- Stops the vehicle motor
- Turns ON safety indicators

## Features
- Real-time drowsiness detection
- IR-based eye blink monitoring
- Automatic vehicle stopping
- Safety alert indicators

## Technologies Used
- Arduino Nano
- Embedded Systems
- IR Sensor
- Motor Control
- C/C++

## Components
- Arduino Nano
- Eye Blink Sensor
- Buzzer
- BO Motor
- BD139 Transistor
- Breadboard
- Jumper Wires
